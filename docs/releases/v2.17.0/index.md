## Key Features in This Release

### Generally Available
- Remote Cluster State Publication
- Disk-Optimized Vector Search
- Faiss Byte Vector Support (HNSW & IVF)
- ML Inference Search Processors
- Batch Asynchronous Ingestion
- Flow Framework User-Level Access Control

### Experimental
- Application-Based Configuration Templates
- Search-Only Replica Shards
- Approximation Framework for Query Optimization
- Custom Trace Source (OpenTelemetry-based)
