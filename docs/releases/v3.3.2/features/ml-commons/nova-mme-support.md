# Nova MME Support

## Summary

Added support for Amazon Nova multimodal embedding (MME) models in ML Commons, enabling users to leverage Nova's multimodal capabilities for embedding generation.

## Details

### What's New in v3.3.2

This is a backport of ml-commons#4360 to the 3.3 branch. It adds support for Amazon Nova multimodal embedding models, allowing users to register and deploy Nova MME models through the ML Commons framework for generating embeddings from multimodal inputs (text and images).
