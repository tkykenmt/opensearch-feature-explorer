# Look & Feel UI Improvements

## Summary

OpenSearch Dashboards Reporting v2.16.0 introduces UI consistency improvements as part of the "Look & Feel" initiative. These changes standardize header and text sizes in reporting menu options to align with OpenSearch Dashboards design guidelines.

## Details

### What's New in v2.16.0

The Look & Feel improvements in v2.16.0 focus on visual consistency:

#### Header and Text Size Adjustments
- Adjusted header sizes in reporting menu options for consistency
- Standardized text sizes across the reporting interface
- Aligned with OpenSearch Dashboards typography guidelines

## Limitations

- Theme-specific font sizes may vary between Next and V7 themes
