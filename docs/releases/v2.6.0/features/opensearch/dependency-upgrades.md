# Dependency Upgrades

## Summary

OpenSearch 2.6.0 introduces 16 new feature(s) and 0 enhancement(s) to Dependency Upgrades, along with 3 bug fixes.

## Details

### New Features

- **Update nebula-publishing-plugin to 19.2.0**: Update nebula-publishing-plugin to 19.2.0
- **Bump `reactor-netty` from 1.1.1 to 1.1.2**: Added in PR #5878.
- **OpenJDK Update (January 2023 Patch releases)**: OpenJDK Update (January 2023 Patch releases)
- **Bump `Mockito` from 4.7.0 to 5.1.0, `ByteBuddy` from 1.12.18 to 1.12.22**: Added in PR #6088.
- **Bump `joda` from 2.10.13 to 2.12.2**: Added in PR #6095.
- **Upgrade to Lucene 9.5.0**: Added in PR #6078.
- **Bump antlr4 from 4.9.3 to 4.11.1**: Added in PR #6116.
- **Bump `Netty` from 4.1.86.Final to 4.1.87.Final**: Added in PR #6130.
- **Bump `Jackson` from 2.14.1 to 2.14.2**: Added in PR #6129.
- **Bump `org.apache.ant:ant` from 1.10.12 to 1.10.13**: Added in PR #6306.
- **Bump `azure-core-http-netty` from 1.12.7 to 1.12.8**: Added in PR #6304.
- **Bump `reactor-netty-http` from 1.0.24 to 1.1.2**: Added in PR #6309.
- **Bump `com.google.protobuf:protobuf-java` from 3.21.12 to 3.22.0**: Added in PR #6387.
- **Bump `io.projectreactor.netty:reactor-netty` from 1.1.2 to 1.1.3**: Added in PR #6386.

- **Bump `jettison` from 1.5.1 to 1.5.3**: Added in PR #6391.

### Bug Fixes

- Bump `re2` and `supertest`
- Upgrade vega-tooltip to `0.30.0` to support custom tooltips
- Add `@opensearch-project/opensearch@^2.x` as dependency aliased as `@opensearch-project/opensearch-next`
