# k-NN

## Summary

OpenSearch 2.6.0 includes 8 bug fixes for k-NN.

## Details

### Bug Fixes

- Remove latestSettings cache from KNNSettings
- Add p99.9, p100 and num_of_segments metrics to perf-tool
- Update bwc to 2.6.0-SNAPSHOT
- Add Windows Support to BWC Tests
- Add test for KNNWeight
- Set NoMergePolicy for codec tests
- Replace KnnQueryVector by KnnFloatVectorQuery for Lucene knn
- Refactor structure of stats module
