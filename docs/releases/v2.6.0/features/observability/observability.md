# Observability

## Summary

OpenSearch 2.6.0 introduces 1 new feature(s) and 0 enhancement(s) to Observability, along with 19 bug fixes.

## Details

### New Features

- **Add new page/route for notebooks create and rename**: Added in PR #250.

### Bug Fixes

- Correct ppl leran more link
- Few fixes regarding issues for visualization rendering
- Debug CVE fix
- Fix bugs related to trace analytics jaeger mode
- Update error handling in viz container
- Panels filter check for where clause
- Fix explorer dark mode issue and restructure scss
- Operational panels cypress fix
- Event Analytics Bug Fixes
- Remove timeseries data validation to get back line chart support
- Add missing changes for visualization config pane
- Move performance now from a debug dependency to a runtime dependency
- Add publish snapshots to maven via GHA
- Add support for structured Metrics & Traces index using Simple Schema for Observability
- Cypress test update and related linting fixes
- upgrade cypress to 6
- Fix Node.js and Yarn installation in CI
- Revert "Removing explicit jackson dependencies
- Bump version to 2.6
